Welcome to the Product Management System

This system was designed to help manage products efficiently. It allows you to create, edit, and delete products in a user-friendly and easy-to-use interface. Built with the Spring Framework and using Thymeleaf for rendering the frontend, this application also integrates with a category system for better product organization.
